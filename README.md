OStreamCatcher
==============

Catches stdout and stderr for ruby blocks.

## Installation

Add this to your Gemfile:

    gem 'o_stream_catcher'

## Example

``` ruby
    require 'o_stream_catcher'

    result, stdout, stderr = OStreamCatcher.catch do
      print "Hello World!"
      42
    end

    result # => 42
    stdout # => "Hello World!"
```
